# Prompt: Crear Modulo CRUD

Actua como Architect, Backend, Frontend, Database y QA.

Objetivo: crear un modulo CRUD Laravel/Livewire siguiendo la arquitectura del proyecto.

Antes de programar:

- Lee `docs-ai/01`, `02`, `04`, `05`, `06`, `09`, `12` y `13`.
- Propón plan de archivos.

Debe incluir:

- Rutas bajo el prefijo correcto.
- Controlador o componente Livewire.
- Modelo Eloquent.
- Migracion si aplica.
- Form Request si aplica.
- Vistas o componentes.
- Permisos si aplica.
- Checklist de pruebas.

No hacer:

- SQL en vistas.
- Logica de negocio en Blade.
- Cambios masivos no relacionados.
