# Prompt: Revisar Codigo

Actua como Tech Lead, Security y QA.

Revisa el cambio con foco en:

- Bugs.
- Seguridad.
- Validaciones.
- Permisos.
- Arquitectura Laravel.
- Duplicacion.
- Consultas ineficientes.
- Responsive si hay UI.
- Pruebas faltantes.

Formato:

- Hallazgos por severidad.
- Archivo afectado.
- Riesgo.
- Recomendacion.
- Pruebas sugeridas.

Si no hay hallazgos, indicar riesgos residuales y pruebas pendientes.
