# Modulos del Dashboard

## Pedidos

Gestion completa del pedido:

- Listado.
- Detalle.
- Estados.
- Asignacion de repartidor.
- Cancelacion.
- Historial.

## Clientes

Registro y consulta:

- Datos del cliente.
- Direcciones.
- Historial de pedidos.
- Estado de cuenta.

## Categorias

Categorias para:

- Comidas.
- Bebidas.
- Promociones.
- Futuras categorias.

## Negocios afiliados

Empresas afiliadas:

- Datos generales.
- Horarios.
- Estado abierto/cerrado.
- Productos asociados.

El negocio afiliado puede gestionar solo su propia informacion, carta, productos, categorias, fotos, horarios y promociones.

No puede gestionar pedidos globales, clientes, pagos, repartidores, usuarios, configuracion, reportes generales ni otros negocios.

## Productos

Carta/productos por negocio afiliado:

- Nombre.
- Precio.
- Categoria.
- Disponibilidad.
- Imagen cuando aplique.

## Repartidores

Gestion y disponibilidad:

- Datos del repartidor.
- Estado disponible/ocupado.
- Pedidos asignados.
- Historial.

## Pagos

Verificacion Yape/Plin:

- Voucher.
- Estado.
- Aprobacion.
- Rechazo.
- Auditoria.

## Reportes

Reportes de:

- Ventas.
- Pedidos.
- Pagos.
- Negocios afiliados.
- Repartidores.

## Usuarios

Operadores/admin:

- Crear usuarios.
- Roles.
- Permisos.
- Activar/desactivar.

Roles oficiales:

- SuperAdmin.
- Admin.
- Operador.
- Negocio Afiliado.
- Repartidor.
- Cliente.

## Configuracion

Datos generales del sistema:

- Nombre.
- Contacto.
- Zonas.
- Tarifas.
- Parametros operativos.
