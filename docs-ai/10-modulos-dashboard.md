# Modulos del Dashboard

## Pedidos

Gestion completa del pedido:

- Listado.
- Detalle.
- Estados.
- Asignacion de repartidor.
- Cancelacion.
- Historial.

## Clientes

Registro y consulta:

- Datos del cliente.
- Direcciones.
- Historial de pedidos.
- Estado de cuenta.

## Categorias

Categorias para:

- Comidas.
- Bebidas.
- Promociones.
- Futuras categorias.

## Restaurantes

Empresas afiliadas:

- Datos generales.
- Horarios.
- Estado abierto/cerrado.
- Productos asociados.

## Productos

Carta/productos por restaurante:

- Nombre.
- Precio.
- Categoria.
- Disponibilidad.
- Imagen cuando aplique.

## Repartidores

Gestion y disponibilidad:

- Datos del repartidor.
- Estado disponible/ocupado.
- Pedidos asignados.
- Historial.

## Pagos

Verificacion Yape/Plin:

- Voucher.
- Estado.
- Aprobacion.
- Rechazo.
- Auditoria.

## Reportes

Reportes de:

- Ventas.
- Pedidos.
- Pagos.
- Restaurantes.
- Repartidores.

## Usuarios

Operadores/admin:

- Crear usuarios.
- Roles.
- Permisos.
- Activar/desactivar.

## Configuracion

Datos generales del sistema:

- Nombre.
- Contacto.
- Zonas.
- Tarifas.
- Parametros operativos.
